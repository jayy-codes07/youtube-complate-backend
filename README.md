#  YouTube Video Platform Backend (REST API)

A fully functional, robust backend REST API for a YouTube-style video hosting and sharing platform. Built with **Node.js, Express.js, and MongoDB**, this architecture handles complex database relationships and secure media file delivery.

###  Tech Stack
* **Core:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Authentication:** JSON Web Tokens (JWT), bcrypt
* **Media Storage:** Cloudinary, Multer (File handling)

###  Key Features
* ** Secure Authentication:** User signup, login, and secure route protection using JWT.
* ** Video Management:** Seamless video uploading, publishing, updating, and deleting with Cloudinary integration.
* ** Community Engagement:** Full support for likes, comments, and a Twitter-style "tweets" system for users.
* ** Subscriptions:** Logic for users to subscribe/unsubscribe to channels.
* ** Playlists & History:** Endpoints to create custom video playlists and track user watch history.
* ** Creator Dashboard:** Aggregated channel statistics including total views, subscribers, and video metrics.
